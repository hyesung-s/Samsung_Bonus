계산식

적용한 공식

  1. OPI1 = 본인 연봉 × OPI1 비율
  2. 사업부별 OPI2 재원 = max(사업부별 예상 영업이익, 0) × OPI2 재원 비율
  3. OPI2 부문 재원 = 전체 OPI2 재원 × 부문 배분 비율
  4. 부문 배분 가중인원 = 메모리 + 파운드리 + S.LSI + 공통 인원 × 공통 가중치
  5. 일반 사업부 OPI2 부문 = OPI2 부문 재원 ÷ 부문 배분 가중인원
  6. 공통 OPI2 부문 = 일반 사업부 OPI2 부문 × 공통 가중치
  7. OPI2 사업부 재원 = 각 사업부 OPI2 재원 × 사업부 배분 비율
  8. 공통 제원 공유 사업부의 사업부 배분 가중인원 = 해당 사업부 인원 + 공통 인원 × 공통 가중치
  9. 공통 OPI2 사업부 = 선택 사업부의 OPI2 사업부 기준금액 × 공통 가중치
  10. 본인 OPI2 = 평균연봉 기준 OPI2 × 본인 연봉 ÷ 평균연봉
