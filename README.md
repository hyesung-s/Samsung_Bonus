# 2026 삼성전자 성과급 계산기

사용자가 입력한 가정값으로 OPI1/OPI2 성과급을 계산하는 비공식 정적 웹앱입니다. GitHub Pages에 그대로 올려 사용할 수 있습니다.

## 기본 가정

- 사업부: 메모리, 파운드리, S.LSI, 공통
- OPI1: 본인 연봉의 50%
- OPI2 재원: 사업부별 예상 영업이익의 10.5%
- OPI2 배분: 부문 40%, 사업부 60%
- 공통: 기본적으로 메모리 제원을 공유하며, 공통 인원 × 70%를 계산에 반영
- 평균연봉: 8,000만 원
- 기본 예상 영업이익: 메모리 320조 원, 파운드리 0조 원, S.LSI 0조 원
- 기본 인원: 메모리 27,400명, 공통 29,000명, 파운드리 14,000명, S.LSI 6,900명

## 이번 버전 반영사항

- 모바일에서 화면 축소 없이 전체 레이아웃이 보이도록 반응형 구조 개선
- 개인조건 입력을 상단에 배치
- 사업부별 예상 영업이익과 공통 분배 가중치 설정을 하나의 표로 통합
- 공통 행에서 제원을 공유할 사업부와 가중치 직접 선택 가능
- 사업부별 예상 성과급 표와 시각화를 통합해 OPI1, OPI2 부문, OPI2 사업부, OPI2 합계를 한눈에 비교
- 요약 카드에 각 항목의 연봉 대비 비율 표시

## 계산 방식

- OPI1 = 본인 연봉 × OPI1 비율
- 사업부별 OPI2 재원 = max(사업부별 예상 영업이익, 0) × OPI2 재원 비율
- OPI2 부문 재원 = 전체 OPI2 재원 × 부문 배분 비율
- 부문 배분 가중인원 = 메모리 인원 + 파운드리 인원 + S.LSI 인원 + 공통 인원 × 공통 가중치
- 일반 사업부 OPI2 부문 = OPI2 부문 재원 ÷ 부문 배분 가중인원
- 공통 OPI2 부문 = 일반 사업부 OPI2 부문 × 공통 가중치
- OPI2 사업부 재원 = 각 사업부 OPI2 재원 × 사업부 배분 비율
- 공통 제원 공유 사업부의 사업부 배분 가중인원 = 해당 사업부 인원 + 공통 인원 × 공통 가중치
- 공통 OPI2 사업부 = 선택 사업부의 OPI2 사업부 기준금액 × 공통 가중치
- 본인 OPI2 = 평균연봉 기준 OPI2 × 본인 연봉 ÷ 평균연봉

## GitHub Pages 배포 방법

1. 이 폴더의 파일을 GitHub repository 루트에 업로드합니다.
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
2. GitHub repository에서 **Settings → Pages**로 이동합니다.
3. Source를 **Deploy from a branch**로 선택합니다.
4. Branch를 `main`, folder를 `/root`로 선택합니다.
5. 저장하면 GitHub Pages 주소가 생성됩니다.

## 로컬 실행

브라우저에서 `index.html`을 직접 열면 됩니다. 별도의 빌드 과정이나 서버가 필요 없습니다.
