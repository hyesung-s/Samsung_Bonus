# 2026 삼성전자 성과급 계산기

정적 HTML/CSS/JavaScript만으로 동작하는 2026 삼성전자 성과급 시뮬레이터입니다. GitHub Pages에 그대로 업로드해 사용할 수 있습니다.

## 포함 기능

- 메모리, 파운드리, S.LSI, 공통 4개 조직 반영
- OPI1 = 연봉 × OPI1 비율
- OPI2 부문 재원 = 사업부별 예상 영업이익 합계 × OPI2 재원비율 × OPI2 부문 배율
- OPI2 사업부 재원 = 사업부별 흑자 영업이익 × OPI2 재원비율 × 사업부 배분 비율
- 공통은 선택한 사업부 재원에 공통 인원 × 공통 가중치를 더해 계산하는 재원보존형 방식
- 본인 연봉, 평균연봉, 사업부별 영업이익, 인원, OPI 조건 수정 가능
- OPI1/OPI2 세전·세후 간이 계산
- OPI2 세후 금액 기준 자사주 수량 계산
- 매도제한 기간별 주식 수 표시: 즉시 / 1년↓ / 2년↓
- 세후 최종 수령액 표시: 현금 OPI1 + 자사주 OPI2 종가 평가액
- 내년도 국민연금/건강보험 납부액 계산: 세전 총소득(연봉 + OPI1 + OPI2) 기준 국민연금 4.75%, 건강보험 3.595% 적용
- VWAP/종가 수동 입력 및 자동 조회 버튼
- 설정값 localStorage 저장 및 URL 공유

## 단위

- 영업이익/재원: 조원
- 성과급 지급액: 억원
- 연봉: 만원
- 주식 가격: 원

## v17-insurance 변경사항

- 세전 총소득을 연봉 + OPI1 + OPI2로 계산합니다.
- 내 결과에 내년도 국민연금과 건강보험의 연간 총금액, 월별 납부액을 추가했습니다.
- 사업부별 예상 성과급 비교에도 사업부별 내년도 국민연금/건강보험 연간 총금액과 월별 납부액을 추가했습니다.
- PC에서는 내 결과 카드를 한 행 중심으로 표시하고, 모바일에서는 가로 스크롤 없이 한 열로 쌓이도록 유지했습니다.
- 모바일에서는 사업부별 예상 성과급 비교 카드가 한 열로 표시되도록 유지했습니다.
- 모바일에서 사업부별 가정 표가 너무 작게 보이지 않도록 행별 카드형 레이아웃을 유지했습니다.

## GitHub Pages 배포

1. 이 폴더 안의 파일을 GitHub 저장소 루트에 업로드합니다.
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
2. GitHub 저장소에서 `Settings` → `Pages`로 이동합니다.
3. Source를 `Deploy from a branch`로 선택합니다.
4. Branch는 `main`, folder는 `/root`를 선택합니다.
5. 생성된 GitHub Pages 주소로 접속합니다.

## 주의

이 계산기는 사용자가 입력한 가정값으로 계산하는 시뮬레이터입니다. 실제 회사 지급 기준, 원천징수, 근로소득공제, 소득공제, 세액공제, 4대보험 산정 기준, 비과세 항목, 실제 자사주 지급 방식과 다를 수 있습니다.
